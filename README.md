### **Frontend Setup**
1. Navigate to the frontend directory:
   
   cd ../frontend
   ```
2. Install dependencies:

   npm install
   ```
3. Start the frontend application:
   `
   npm start






### **Backend Setup**
1. Navigate to the backend directory:
   
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file 
   
  

4. Start the backend server:
   
   npm start
   ```
